>1. `google-fonts`<br>
    该资源包含900多个material design设计系统所用的全部图标，多种尺寸选择，还包含可实际运用的网页字体,可以在网页、Android和iOS项目中使用。 不仅支持色彩，位置，大小自定义操作，而且该资源完全免费。
>2. `@mdi/font`<br>
    Ant Design 是一个 UI 设计语言，服务于企业级产品的设计体系，里面也有单独的图标设计篇章，安装 [Kitchen Sketch](https://kitchen.alipay.com/)插件可以一键拖拽使用 Ant Design 和 Iconfont 的海量图标，还可以关联自有项目。<br>
    基本用法:`npm install --save @ant-design/icons`<br>
    [代码演示](https://ant.design/components/icon-cn/#components-icon-demo-basic)
